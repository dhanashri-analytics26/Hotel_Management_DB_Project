
-- Hotel Management Database Project

-- Project Overview
-- This project is a SQL based Hotel Management Database System
-- designed to manage hotel details and customer reviews in a
-- structured and efficient manner.
-- The project demonstrates database design, data integrity,
-- CRUD operations, testing, and documentation using
-- industry style practices.
-- This project is suitable for SQL Data Analyst and
-- Business Analyst fresher roles.

-- -----------------------------------------------------
-- Business Problem
-- -----------------------------------------------------
-- Hotels need a reliable system to store hotel information
-- and customer feedback.
-- Manual or unstructured data handling can lead to
-- data inconsistency, errors, and difficulty in reporting.

-- -----------------------------------------------------
-- Solution
-- -----------------------------------------------------
-- A relational database was designed using MySQL with
-- proper normalization and relationships.
-- The solution supports:
-- Hotel master data management
-- Customer review tracking
-- Business reporting using JOIN queries
-- Reusable data retrieval using stored procedures
-- Structured testing using Excel test cases

-- -----------------------------------------------------
-- Tools and Technologies Used
-- -----------------------------------------------------
-- Database        MySQL
-- Tool            MySQL Workbench
-- Testing         MS Excel
-- Documentation   MS Word
